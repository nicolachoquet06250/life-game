# Le jeu de la vie
Développement du jeu de la vie en typescript vanilla

> Le jeu est responsive et entièrement utilisable sur mobile.

Fonctionnalités:
- Zoom + / - via :
  - **Desktop**: molette
  - **Mobile**: geste "pinch"
- Déplacement multidirectionnel via :
  - **Desktop**: click prolongé + slice
  - **Mobile**: slice
- Pause / Play
- Réinitialiser
- **Allumer une cellule** en **cliquant** sur **une cellule éteinte**
- **Éteindre une cellule** en **cliquant** sur **une cellule allumée**
